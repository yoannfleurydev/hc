# Highlight Cat

Small ZSH script to highlight source code using `cat` and `highlight`.

## Installation

* `cd && mkdir $HOME/.zsh -p`
* `git clone git@github.com:yoannfleurydev/hc.git $HOME/.zsh/hc`
* In your `.zshrc`, add `source $HOME/.zsh/hc/hc.zsh`
